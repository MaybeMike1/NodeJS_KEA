Linux and Npm Commands

# Linux Command: Description:
1 ls - list files It stands for list files, so it lists all the files and folders of the directory that you are inside.
2 cd - change directory This command is what you want to remember as it is the most used one. It is used to change directory to another folder.
3 touch [filename] - Creates new file It creates a new file inside the directory, that you are currently in. This can be used to make text files, but can also be used to make .py or .js files
4 cat [filename] It is used to see what is inside the file, that the cat has been called on. It prints the content of the file in the terminal.
5 echo [text] What echo means is, that it reprashe what you echo. You could say the terminal repeats the text you enter.
6 mkdir - Make directory This command makes a new directory or folder inside the current directory, that you are working in.
7 rm [filename] - Remove file Removes the selected file or directory inside the working directory
8 rmdir [directory] - Remove directory Removes the selected directory inside the working directory
9 cp [newOfNewFile] [pathOfFileToCopy] - Copy file Copy a file into the working directory.
# NPM Commands: Description:
1 npm install [package] This is used to install dependecies with Node Package Manager. This can also be translate into npm -i [dependency]
2 npm init -y Creates a package.json file inside the directory, that we are working inside. The flag y is to set it to defualt.
3 npm install -g [package] This is used to install packages globally, so they can be be used in any project. This would be smart to do with 'nodemon'
4 npm ls Lists all installed modules.
