Express is a module, that makes it possible for us developers to run an application with JavaScript (NodeJS). But how do we use that module and where does all these functions associate with Express come from. We use our Node Package Manager (npm) to get it installed. But before we try to install the module, we have to make sure there is a package.json file inside of our project. Because our package.json is where the dependencies are stored. When the json file is instanciated, we can now install our Express module to the project. There are two ways to do this. The two are listed below:

  • The other way is a bit more complicated than the first one, but we talked about the package.json file. In a json file the syntax is as a JavaScript object, so in the json file, there is a key and to that key there is a value associate to that key. So we specify the dependecies and inside the dependecies object, there will be added a key, Express and a value which is the version number of Express.
